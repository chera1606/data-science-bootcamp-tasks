# Data Science Bootcamp Tasks: Python Foundations

## Description

This repository contains solutions for the Python Foundations assignment from the Data Science Bootcamp.  
It includes three tasks across different levels, each implemented in a simple, clear and follows best practices.

---

## **Level 1: Check if a number is prime**

**File:** `prime_check.py`

**Description:**  
This program asks the user to input a number and checks if it is prime.  
A number is considered prime if it is greater than 1 and cannot be divided evenly by any other number except 1 and itself.  
The program loops through numbers from 2 up to the given number and prints whether the number is prime or not.

---

## **Level 2: Count the number of vowels in a string**

**File:** `count_vowels.py`

**Description:**  
This program counts how many vowels (`a, e, i, o, u`) are present in a string provided by the user.  
The program loops through each character of the string, checks if it is a vowel (case-insensitive), and keeps a count.  
Finally, it prints the total number of vowels in the input string.

---

## **Level 3: CLI-based calculator using functions**

**File:** `calculator.py`

**Description:**  
This program is a simple calculator that performs addition, subtraction, multiplication, and division.  
The program uses functions for each operation.  
The user chooses the operation and inputs two numbers.  
The program calls the corresponding function, performs the calculation, and prints the result.  
It also handles division by zero gracefully with an error message.

---

## **How to Run**

1. Make sure Python is installed on your system.
2. Open the terminal or VS Code terminal in this project folder.
3. Run any program using:

```bash
python file_name.py
```
